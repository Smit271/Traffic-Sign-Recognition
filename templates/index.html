What is Image Recognition in Neural Networks ?

Image recognition is one of the tasks in which deep neural networks (DNNs) excel. Neural networks are computing systems designed to recognize patterns. Their architecture is inspired by the human brain structure, hence the name. They consist of three types of layers: input, hidden layers, and output layer

What is this project about?

This is a mini project which I build in my 7th semester of my computer engineering study. I have built Neural Networks model which classifies the traffic signs as user give input to the model. I have used CNN classification algorithm and GTSRB - German Traffic Sign Recognition Benchmark dataset from kaggle.

How I built this project ?

How I built this project ?

Below i described How I implemented CNN to make model.

Step 1 : Downloading the dataset

I have used GTSRB - German Traffic Sign Recognition Benchmark dataset to train my model. You can explore more about data by click on it if you want to understand data.

Step 2 : Installing necessary libraries and importing them

Most important framework is Tensorflow which provides platform for building neural network models easily and conveniently

Step 3 : Loading and preprocess the data

Very important factor in computer vision application of neural networks is to make sure yout have scaled and same distributed type data.

Step 4 : Creating Model Architecture

In this part of implementation, I created Conv layers followed by Pooling layers type architecture.

Step 5 : Compile and train the model with suitable HyperParameters

After setting HyperParameters for 2-3 days i finally made best model with 95.12 % Training accurcay, 98.80 % Validation accurcay and 94.48 % Testing accuracy which best in my case of creating models.
